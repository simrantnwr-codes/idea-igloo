Step 1: The Input

What Counts as a Signal?

A signal is any external change that creates new pain, urgency, or opportunity. If you can't link to a source (news, policy, API update), it's not a signal.

Valid Signals (Fuel)

USE THESE
✓
Regulatory Changes

Gov policies, RBI circulars, Health Ministry guidelines, DPDP Act.

✓
Platform Shifts

Google Algo updates, WhatsApp API changes, Apple Privacy rules.

✓
Market Shifts

Layoffs (freelancer supply), funding dry-ups (cost-cutting focus).

✓
Tech Inflections

New API capabilities, drastic cost drops (e.g., token costs).

Invalid Signals (Distraction)

AVOID
✕
Vague Trends

"AI is growing" or "People care about health" without a specific trigger.

✕
Pure Speculation

Ideas based on technology that doesn't exist yet.

✕
Lagging Indicators

Old news (e.g., "Remote work is rising" - 2020 news).

✕
No Evidence

"I feel like people need this" without validation.

Why this order matters
